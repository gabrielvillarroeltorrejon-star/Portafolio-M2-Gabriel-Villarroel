# Portafolio-M2-Gabriel-Villarroel
Tour Clima
¿Qué hace esta página?
Lista de destinos: Muestra 10 destinos ubicados entre la region metropolitana y 5° y sus respectivo clima.
Tarjetas interactivas: Al pasar el mouse, las tarjetas se elevan para indicar que puedes hacer clic.
Vista de detalles: Al elegir un destino, la pantalla cambia para mostrar la humedad, el viento y el pronóstico de toda la semana.
Botón volver: Permite regresar a la lista principal sin recargar la página.
Herramientas usadas
-HTML y CSS: Para la estructura y los colores.
-Bootstrap: Para que la página se vea bien en celulares y computadoras.
-JavaScript: Para que la información cambie de forma dinámica cuando haces clic.
